TPT¶

In this module, the Transition Path Theory (TPT) analysis is implemented. [NSVE+09]

tpt namelist¶

The tpt namelist has following parameters:

Necessary Parameters

tpmfile

character The transition probability matrix file which has the format of (ncluster * ncluster)

statefile

character The state index file

startstate

integer The index of start state of runing TPT analysis

endstate

integer The index of end state of runing TPT analysis

Example¶

The example of tpt namelist shows below

&tpt
    tpmfile = "tpm.txt"
    statefile = "states.txt"
    startstate = 10
    endstate = 8
/

Output File Format¶

tpt.out Transition path theory results, it’s the net flux matrix from start state to end state which has a form of (number of state * number of state)

Run the cluster command¶

Default input file is “tpt.in”，and default output file is “tpt.out” The user can provide file by -i with input file, and -o with output file.

./msm.x tpt
./msm.x tpt -i file.in -o file.out
