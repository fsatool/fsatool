EDS (Essential dynamic sampling)¶

Doing the conformation sampling using the EDS method. This method needs a pre-simulation to get the data of eigenvalue, eigenvector to run the following simulation. The EDS could be implemented in one collective variable space or two collective variables space according the input file. The input file of EDS needs a eds namelist.

EDS namelist¶

edsdis0

double precision The threshold of the distance between coordinates in the first CV eigenvector space and reference coordinates in the first CV eigenvector space. When the distance less than this threshold, then this round of EDS is finished.

maxpathstep

integer The max step in a round of EDS

pcstart

integer The start index of principal component to run EDS in the first CV space

pcend

integer The end index of principal component to run EDS in the first CV space

eigenfile

array of character The data of eigenvalue, eigenvector of the pre-simulation in the first CV.

edsatoms

array of integer The needed atom index of running EDS in the first CV

edsmdstep

integer The number of MD step between EDS of first CV space and ENS of second CV space

edsfixstep

integer The number of step of running EDS in the first CV space

edsfixstep2

integer The number of step of running EDS in the second CV space

edsdis02

dobule precision The threshold of the distance between coordinates in the second CV eigenvector space and reference coordinates in the second CV eigenvector space. When the distance less than this threshold, then this round of EDS is finished.

pcstart2

integer The start index of principal component to run EDS in the second CV space

pcend2

integer The end index of principal component to run EDS in the second CV space

eginefile2

array of character The data of eigenvalue, eigenvector of the pre-simulation in the second CV.

edsatoms2

array of integer The needed atom index of running EDS in the second CV space

Input File Format¶

  • pca_eigenvecs

refcoor: The reference coordinate of edsatoms

xi0: The projection of reference coordinates to the eigenvector space

eigenval: The eigenvalue of the pre-simulation trajectory

1: The correspond eigenvector to the first eigenvalue

2: The second eigenvector to the second eigenvalue

Output File Format¶

  • eds_path_1.traj

The Cartesian coordinate of atoms in EDS sampling, the number of trajectory file is determined by the total steps and the number of steps in a round of EDS

Additional Information about EDS¶

  1. When using the Mixing-REMD method, the ifeds in the task namelist should be set to TRUE.

  2. When you need to de the EDS in only one CV space, the variable of EDS namelist behind the edsmdstep could be removed.
